# 🖼️ Simple Image Slider

This is a simple and responsive image slider built using **Vanilla JavaScript (ES6)**.  
It allows users to navigate between images using "Previous" and "Next" buttons, and also provides numbered indicators.

📸 Demo
🔗 Live Preview – https://lo22op.github.io/Slider/

## 🚀 Features

- Navigate through images using buttons.
- Display current image number (`#1 Of N`).
- Dynamically generated indicators (`<ul>` with `<li>`) for each slide.
- Active image and active indicator are highlighted.
- Disables navigation buttons when reaching the first or last slide.
- Built using modern JavaScript techniques (`Array.from`, event listeners).
- Clean structure and easy to understand.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
